# University Management API

REST API for managing students, teachers, courses and enrollments.

## Technologies

- Java 21
- Spring Boot
- Spring Data JPA
- H2 Database
- Swagger/OpenAPI
- JUnit 5

## Features

- CRUD for Students
- CRUD for Teachers
- CRUD for Courses
- CRUD for Enrollments
- Student search
- Student filtering
- Course filtering
- GPA calculation
- Unpaid enrollments endpoint
- Global exception handling
- Unit tests

## Run

```bash
./gradlew bootRun
```

Swagger UI:

```
http://localhost:8080/swagger-ui/index.html
```
